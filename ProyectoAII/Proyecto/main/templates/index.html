Bienvenido al Sistema comparador de precios de claves de videojuegos!!!

Se ha construido un sistema web que permite buscar precios de claves de videojuegos en distintas paginas

Las páginas usadas son:

    Instant-Gaming

    G2A

    Eneba

Ademas de esto se ha implementado un sistema de recomendación basado en contenido, de forma que si

el videojuego buscado se encuentra en la base de datos se muestran otros juegos que quizás le puedan

interesar al usuario. Para construir este sistema de recomendación se usa un dataset con mas de 27000

juegos presentes en steam clasificados por categorías.
